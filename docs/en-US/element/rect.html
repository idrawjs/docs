矩形元素 ​

Rectangle ​

It is used to render rectangle content. The specific type is as follows:

ts
type Rect = {
  type: 'rect';
  x: number;
  y: number;
  w: number;
  h: number;
  angle: number;
  detail: {
    background: string;
    borderRadius?: string;
    borderWidth?: number;
    borderColor?: number;
  };
};

Rect.detail ​

The basic attribute details of the element can be viewed Element's Detail # .

Rect.detail Data-Type ​

The Data-Type of Rect.detail

ts
type RectDetail = {
  background?: string;
  background?: string;
  borderWidth?: number;
  borderColor?: string;
};
