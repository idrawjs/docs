# SVG元素

# SVG

用来渲染SVG代码片，具体格式如下所示：

type SVG = {
  type: 'svg',
  x: number,
  y: number,
  w: number,
  h: number,
  angle: number,
  desc: {
    svg: string,
  }
}

# SVG.desc 详解

元素的基本属性详情可以看 元素介绍-数据基本格式 。

# SVG.desc 数据格式

SVG 元素的desc属性

type SVGDesc = {
  svg: string,
}

# SVG.desc 数据属性内容

属性 说明 类型 默认值 是否必填 备注
svg SVG代码片段 string - true -
